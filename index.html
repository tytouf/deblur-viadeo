Deblur Viadeo

This one weird bookmarklet makes all blurry texts readable.

This can be used to deblur the blurry texts found on Viadeo web pages. It appears that the text is actually in clear in the html file but was simply blurred using CSS properties. This bookmarklet can be used to remove these CSS properties. To Install it, drag the button below in your browser toolbar. To use it, simply click the bookmark when you are visiting a Viadeo page hiding text with blur.

Deblur Viadeo

Drag button above to your browser toolbar to install.
